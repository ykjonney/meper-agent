宿主页（模拟第三方网站）— chat-widget.js + data-api-key

第三方接入只需一行 <script>：API Key 写在 data-api-key， chat-widget.js 在 iframe 内 client 启动时经 postMessage 把 key 注入（client 自身不内置 key）。 
<script src="http://localhost:3001/chat-widget.js"
        data-chat-url="http://localhost:3001"
        data-api-key="af_live_xxx"
        data-title="Agent Flow 助手"
        data-width="680px"></script>
可选：data-user-token（callback 模式的 X-User-Token）、data-width/right/bottom/open-on-load。 JS API：AgentFlowChat.open() / .close() / .toggle()。
