# Product

## Register

product

## Users

技术人员（开发者、运维）和非技术管理者混合使用。技术人员在工作站上长时间操作 Agent、工作流和任务；管理者偶尔查看执行状态和统计数据。需要兼顾易用性和信息深度。

## Product Purpose

Agent Flow 是一个 AI Agent 编排平台，用于创建、配置、部署和监控 AI Agent 与工作流。核心价值：让团队可视化地构建和管理多 Agent 协作流程，从单 Agent 对话到多步骤工作流编排、任务调度和执行监控。

## Brand Personality

专业 · 现代 · 有活力 · 工程感

不是冰冷的纯工具感，也不是花哨的营销风。像一个靠谱的技术搭档：精确、高效，但不无聊。通过排版节奏、精确的色彩点缀和流畅的微交互传达品质感。

## Anti-references

- Dify 的卡通插画风（太 casual，不适合 B 端工业场景）
- Vercel 的纯黑冷峻风（太戏剧化，不适合长时间使用）
- 传统 AntD Pro 后台（老式 B 端感，缺乏个性）
- 通用 SaaS 模板的奶油色暖调（2026 AI 泛滥风格）

## Design Principles

1. **信息层次即功能** — 通过排版密度和视觉权重的差异，让用户本能地区分"需要关注的"和"背景信息"
2. **状态永远可见** — Agent 状态、执行进度、错误信息必须有即时的视觉表达，永远不模糊
3. **克制但不无聊** — 单色系主色调 + 精准的语义色点缀；动效服务于反馈而非装饰
4. **工程内容为第一公民** — 代码、ID、日志、JSON 用等宽字体正确渲染，不降级对待
5. **密度可调** — 表格和列表支持紧凑模式；卡片布局保持呼吸感但不浪费空间

## Accessibility & Inclusion

- WCAG 2.1 AA 级别
- 正文对比度 ≥ 4.5:1，大文本 ≥ 3:1
- 所有交互支持键盘导航
- 动画支持 `prefers-reduced-motion` 降级
